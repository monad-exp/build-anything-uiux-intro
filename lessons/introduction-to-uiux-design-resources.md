# Resources

- [Nielsen Norman Group: UX vs UI](https://www.nngroup.com/articles/what-is-user-experience/) - how UX and UI differ, including what UX is and is not
- [Nielsen Norman Group: User-Centered Design](https://www.nngroup.com/topic/user-centered-design/) - how to design around real user needs and goals
- [Interaction Design Foundation: UX Design](https://www.interaction-design.org/literature/topics/ux-design) - foundational overview of user experience design
- [Interaction Design Foundation: Information Architecture](https://www.interaction-design.org/literature/topics/information-architecture) - how to structure content and functionality clearly
- [Google Material Design](https://m3.material.io/) - design system reference for layout, components, and patterns
- [Apple Human Interface Guidelines](https://developer.apple.com/design/human-interface-guidelines/) - platform design principles and interface patterns
- [WCAG Accessibility Guidelines](https://www.w3.org/WAI/standards-guidelines/wcag/) - standards for contrast, labels, keyboard navigation, and screen readers
- [WebAIM](https://webaim.org/) - practical accessibility guidance and tools
- [Usability.gov](https://www.usability.gov/) - research-backed guidance on usability and information architecture
- [The Design of Everyday Things](https://www.nngroup.com/books/design-everyday-things-revised/) - classic introduction to human-centered design thinking
