# Resources — Introduction to UI/UX Design for Web3

Supplementary links for concepts, products, and practices referenced in this lesson.

## Web3 Products to Study

These products are called out in the lesson as examples of strong UI/UX. Explore how they handle wallet connection, transaction states, and complex actions.

- [Uniswap](https://app.uniswap.org) — Decentralized exchange; study swap flows, approvals, and transaction feedback.
- [Phantom](https://phantom.app) — Wallet app; study onboarding, connection patterns, and clear transaction signing.
- [Coinbase](https://www.coinbase.com) — Consumer crypto platform; study trust signals, simplified language, and guided flows.
- [Jupiter](https://jup.ag) — Solana swap aggregator; study dense trading UI made approachable for everyday users.

## Creating the Interface

### Visual Hierarchy & Consistency

- [Visual Hierarchy in UX Definition](https://www.nngroup.com/articles/visual-hierarchy-ux-definition/) — Making important elements (APY, balances, primary actions) impossible to miss.
- [Design Systems Gallery](https://designsystems.surf/) — Reference for consistent buttons, spacing, patterns, and component behavior across a product.

### Accessibility

- [Web Content Accessibility Guidelines (WCAG)](https://www.w3.org/WAI/standards-guidelines/wcag/) — Standards for color contrast, readable text, labels, keyboard navigation, and screen reader support.
- [WebAIM Contrast Checker](https://webaim.org/resources/contrastchecker/) — Quick tool to verify text and background contrast.
- [The A11Y Project](https://www.a11yproject.com/) — Practical accessibility checklists and patterns for web interfaces.
